Description du projet
======================
Exercice de création d'un repository git

Créer un repository git sur machine locale
Ajouter un fichier README.txt pour décrire le projet
Apporter une modification à ce texte
Ajouter un autre fichier texte
Ajouter un dernier fichier texte
Faire un commit à chaque modif.

Le repository local sera ensuite partagé sur GitHub
